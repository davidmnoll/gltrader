# Setup Instructions For Mac



##To Run:

## 1: CD into base directory of install

    cd C:\\Path\\To\\project\\dir\\gltrader

## 2: Activate venv

    venv\\Scripts\\activate

## 3: Run Module

    python -m gltrader

    cd C:\\Path\\To\\project\\dir\\gltrader

## 2: Activate venv

    venv\\Scripts\\activate

    python -m nosetests
